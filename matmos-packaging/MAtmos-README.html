MAtmos

Quick info

  • ModLoader must be installed.
  • MAtmos doesn't alter any base classes.
  • MAtmos is pretty much compatible with any other mod.
  • MAtmos will perfectly run on public servers regardless of whether the server has MAtmos installed or not.

Installing

  • Make sure ModLoader is installed.
  • Open .minecraft/ folder
  • Open MAtmos_r__x_y_z.zip (the file you've downloaded)
  • Select everything in the archive
  • Drag and drop all the selected files into the .minecraft/ folder
  • IF YOU DO THIS, DO NOT PUT THE FILES IN THE MINECRAFT.JAR FOR PETE'S SAKE!

Customizing MAtmos

In weapon_interactions, add sword/item sounds for custom items from other mods

Must-do

  • Figure out the item ID of the custom item
  • Open matmos_expansions_r7/ folder
  • Remove interactions_weapons.xrl to prevent conflicts
  • Open matmos_internal/storage folder
  • Copy interactions_weapons.xrl.xmlo
  • Paste it in matmos_expansions_r7/
  • Rename the file you've just pasted to "interactions_weapons_custom.xml"

Using a text editor

  • Open the file "interactions_weapons_custom.xml" with a text editor
  • Observe the lists between line 2 and line 91
  • Find the appropriate list to add the itemID of your custom item
  • Add a constant value to that list using the same scheme, with your custom item ID
  • Save

    • Using MAtmos Designer

  • Open the file "interactions_weapons_custom.xml" with MAtmos Designer
  • Go to the "Lists" tab
  • Find your appropriate list
  • Add a constant value to that list with your custom item ID, following MAtmos Designer instructions
  • Save
